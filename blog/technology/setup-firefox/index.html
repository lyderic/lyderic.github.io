Setup Firefox

8 décembre 2024

After having installed Firefox, these are the steps I take to set it up and configuring it to my liking and needs:

Configuration

  • Theme: Cheers - Bold
  • Zoom: Make Firefox bigger

Extensions

  • uBlock Origin
  • Birwarden
  • Wallabag
  • Vimium
  • Dictionnare français

Restore settings

From the usual configuration directory, restore the following:

  • vimium-options.json
  • wallabag.json

Logins

  • Log in to Bitwarden extension
  • Log in to Firefox sync

Settings

  • ‘Firefox Home Content’ > ‘Shortcuts’: unchecked
  • ‘Default Search Engine’: DuckDuckGo
  • ‘Open previous windows and tabs’: checked
  • ‘Ask to save password’: unchecked
  • ‘Enable DNS over HTTPS using’: ‘Max Protection’
